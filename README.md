# Substrate Node Template

- Link: https://github.com/substrate-developer-hub/substrate-node-template <br/>
- Branch: polkadot-v0.9.26

## Getting Started

```bash
# Build
cargo build --release

# Run
./target/release/node-template --dev

# Test
cargo test -p  PALLETS-PACKAGE-NAME

# fmt
cargo +nightly fmt
```
